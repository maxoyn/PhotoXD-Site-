# PhotoXD
A gallery website using .NET MVC5 </br>
</br>
It has the following features:
 * Responsive design (bootstrap)
 * CRUD with database
 * Account roles (normal user and admin)
 * Async calls - creating, deleting, and showing comments by user or date
